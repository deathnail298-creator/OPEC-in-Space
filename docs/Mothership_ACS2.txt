ACS2 – Mothership Tank Farm Topology (OPEC-in-Space)

NODES:
- DOCK_CO2       : CO₂ skimmer interface at docking cone
- DOCK_FUEL      : Fuel (H₂/CH₄/He-3) skimmer interface at docking cone
- CO2_M          : CO₂ backbone manifold
- O2_M           : O₂ backbone manifold
- FUEL_M         : Fuel backbone manifold (H₂/CH₄/He-3)
- VAC_M          : Vacuum/pressure management manifold
- CO2_TANK_RING  : Inner ring of partitioned CO₂ tanks
- O2_TANK_RING   : Ring of LOX tanks
- H2_TANK_RING   : Ring of LH₂ / high-pressure H₂ tanks
- CH4_TANK_RING  : Ring of CH₄ tanks
- HE3_TANKS      : Small, high-pressure He-3 tanks
- SOXE           : CO₂→O₂ solid oxide electrolysis module
- ENGINE_FEED    : Mothership propulsion feed header
- DEPOT_PORT     : External offload port (cis-lunar / depot customer)
- POWER_BUS      : Primary/secondary/survival power distribution
- THERMAL_ZONE_H : Hot zone (SOXE and associated electronics)
- THERMAL_ZONE_C : Cold zone (cryogenic tank region)
- RCS_BLOCK      : Attitude control thrusters
- AVIONICS_CORE  : Guidance, comms, autonomy

EDGES (all dual-path or dual-lumen where critical):
- DOCK_CO2   → CO2_M
- CO2_M      ↔ CO2_TANK_RING      (multi-path, check/isolation behavior)
- CO2_M      → SOXE               (reactor inlet)
- SOXE       → O2_M               (O₂ outlet)
- O2_M       ↔ O2_TANK_RING       (multi-path)
- O2_M       → ENGINE_FEED
- O2_M       ↔ DEPOT_PORT         (delivery to customers)

- DOCK_FUEL  → FUEL_M
- FUEL_M     ↔ H2_TANK_RING
- FUEL_M     ↔ CH4_TANK_RING
- FUEL_M     ↔ HE3_TANKS
- FUEL_M     ↔ DEPOT_PORT         (fuel delivery)

- VAC_M      ↔ CO2_TANK_RING      (redundant vacuum interfaces)
- VAC_M      ↔ O2_TANK_RING
- VAC_M      ↔ H2_TANK_RING
- VAC_M      ↔ CH4_TANK_RING

- POWER_BUS  → SOXE
- POWER_BUS  → AVIONICS_CORE
- POWER_BUS  → RCS_BLOCK
- POWER_BUS  → thermal control elements in THERMAL_ZONE_H and THERMAL_ZONE_C

- THERMAL_ZONE_H → radiators (via heat pipes)
- THERMAL_ZONE_C ↔ insulation/MLI, limited conductive links to structure
